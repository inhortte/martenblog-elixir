Grid System

Build fully responsive, and adaptive grid layouts.

To create a responsive grid system you'll need a .container class and a .row-fluid class.
Here's how you'd set up a grid system with a four column col4 and an eight column col8 layout. 

<div class="container">
    <div class="row-fluid">
      <div class="col4">
         ... 
      </div>
      <div class="col8">
         ... 
      </div>
    </div>
</div>

One Block Of Code

No need to keep track of two separate stylesheets for your responsive grid system. You can edit your css for different breakpoints all in one place.

.rem is a function created to turn pixels into REM's. We explain how to use it in mixins.less. 

.col12 {
  /* Laptop Screens */
  .rem(width, 940);
  }
  
  @media @bp-desktop {
    /* Large Desktop screens */ 
    .rem(width, 1170);
  } 
  
  @media @bp-tablet { 
    /* Tablet screens */
    .rem(width, 724);
  }
  
  @media @bp-mobile {
    /* Mobile screens */
    width: 100%;
  }
}
